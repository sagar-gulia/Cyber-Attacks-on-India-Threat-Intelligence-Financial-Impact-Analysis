# Cyber-Attacks-on-India-Threat-Intelligence-Financial-Impact-Analysis

📌 Project Overview
Developed an interactive Power BI dashboard analyzing cyber attacks targeting India
Focused on attack sources, types, and financial impact


📊 Key Features
Visualizes total cyber attacks, financial losses, and prevention costs
Identifies top source countries contributing to attacks and losses
Analyzes distribution of attack types (Phishing, Malware, Ransomware, etc.)
Sector-wise analysis of financial impact across industries
Time-based trends showing yearly attack patterns and loss fluctuations


🎛️ Interactivity
Implemented dynamic slicers (filters):
Year
Source Country
Attack Type
Severity Level
Target Sector
Enables multi-dimensional data exploration

🔍 Reference Sources
CERT-In – Cyber incident reports in India
National Crime Records Bureau – Cybercrime statistics
IBM – Cost of Data Breach reports
Kaspersky – Global threat intelligence reports
Statista – Cybersecurity and financial loss estimates


💰 Financial Insights
Compares Estimated Loss vs Prevention Cost
Highlights high-risk sectors and critical severity attacks
Identifies countries causing maximum financial damage
🛠️ Tools & Technologies
Microsoft Power BI – Data visualization & dashboard creation
Microsoft Excel – Data preparation and dataset creation


📈 Outcome
Provides actionable insights into cybersecurity risks and economic impact
Demonstrates skills in data analysis, visualization, and storytelling
